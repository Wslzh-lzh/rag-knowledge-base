## 项目概览

企业级知识库问答系统（RAG），包含 FastAPI 后端、Next.js 前端、PostgreSQL/Redis/Qdrant/OpenSearch/MinIO 基础设施，支持文档入库、混合检索、引用回溯和流式问答。

### 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Next.js 14 + React 18 + TypeScript + Tailwind CSS |
| 后端 | FastAPI + SQLAlchemy 2.0 + Pydantic v2 + Alembic |
| 数据库 | PostgreSQL 16（主库）+ SQLite（本地开发） |
| 缓存 | Redis 7 |
| 向量检索 | Qdrant |
| 全文检索 | OpenSearch |
| 对象存储 | MinIO（兼容 S3） |
| 认证 | JWT (PyJWT + passlib/bcrypt) |
