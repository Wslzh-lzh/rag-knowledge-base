## 4️⃣ 动态效果（Micro-interactions）—— 让页面活起来

**提示词：**

```
Micro-interaction animations for RAG platform UI:

1. Login page: 
   - Floating particles moving slowly upward with varying opacity
   - Login card fades in with scale(0.95) → scale(1) transition (0.6s ease-out)
   - Input fields focus with blue glow border expansion
   - Submit button ripple effect on click + loading spinner state

2. Q&A page:
   - AI response messages typewriter effect with cursor blink
   - Source reference pills fade in sequentially with stagger delay
   - Send button morphs into loading spinner while waiting for response
   - Sidebar item hover: subtle background shift + icon scale

3. Workbench:
   - Metric cards count-up animation from 0 to final number
   - Sparkline charts draw path animation on load
   - Activity feed items slide in from right with stagger
   - Status dots pulse with breathing animation

4. Global:
   - Page transitions with smooth opacity and slight Y-offset
   - Tooltip appears with soft fade and slide-up
   - Scrollbar custom styled with gradient thumb
   - Loading skeleton shimmer effect on content areas

All animations: smooth, purposeful, not overwhelming,
Easing functions: cubic-bezier(0.4, 0, 0.2, 1),
Duration: 200ms - 600ms,
Respects prefers-reduced-motion accessibility
```
