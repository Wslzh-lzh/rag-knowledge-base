## 系统简介

**RAG 知识库问答系统**是一款基于大语言模型的企业级知识管理与智能问答平台。通过 RAG（检索增强生成）技术，将你的文档、知识库与 AI 能力结合，实现：

- 📚 **知识库管理** - 创建、编辑、删除知识库，支持私有/公开权限
- 📄 **文档解析** - 支持 .txt、.md、.html 等多种文本格式自动解析
- 💬 **智能问答** - 基于知识库内容的 AI 对话，带引用来源
- 🔍 **向量检索** - 支持相似度搜索，快速定位相关内容
- 🎨 **玻璃态界面** - 现代化深色主题，流畅的交互动效

### 3. 知识库管理

**访问地址**: http://localhost:3000/kb

知识库是文档的集合，是问答的知识来源。

#### 3.1 创建知识库

1. 点击右上角「**+ 新建知识库**」按钮
2. 填写知识库信息：
   - **名称**（必填）- 知识库的名称
   - **描述**（可选）- 简要说明知识库内容
   - **可见性** - 选择「私有」或「公开」
3. 点击「创建」按钮

> **可见性说明**:
> - **私有** - 只有创建者本人可以访问
> - **公开** - 所有登录用户都可以查看

#### 3.2 知识库列表

- 以卡片形式展示所有知识库
- 显示名称、描述、可见性
- Hover 时显示「删除」按钮
- 点击卡片进入知识库详情页

#### 3.3 删除知识库

1. 将鼠标悬停在知识库卡片上
2. 点击右下角的「删除」按钮
3. 在确认对话框中点击「确定」

> ⚠️ 删除知识库会同时删除其中的所有文档和数据，请谨慎操作！

### 4. 文档上传与管理

**访问方式**: 知识库列表 → 点击某个知识库

进入知识库详情页，可以管理该知识库中的所有文档。

#### 4.1 支持的文档格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| 纯文本 | `.txt` | 纯文本文件 |
| Markdown | `.md`, `.markdown` | Markdown 格式文档 |
| HTML | `.html`, `.htm` | 网页文件，自动提取正文 |

#### 4.2 上传文档

1. 点击「上传文档」区域（虚线框）
2. 选择本地文件
3. 等待上传和解析完成

**上传状态说明**:

| 状态 | 颜色 | 说明 |
|------|------|------|
| **处理中** | 黄色 | 文档正在解析和向量化 |
| **已完成** | 绿色 | 文档已就绪，可以用于问答 |
| **失败** | 红色 | 文档解析失败，请重试 |

#### 4.3 文档列表

- 显示该知识库下的所有文档
- 显示文件名、类型、解析状态
- 点击文档可查看详情

#### 4.4 文档详情弹窗

点击文档后弹出详情窗口，包含两个 Tab：

**📄 文档预览**
- 展示文档的文本内容预览
- 仅显示前 5 个 Chunk 的内容
- 支持滚动查看

**🧩 Chunk 列表**
- 显示文档被切分后的所有 Chunk
- 每个 Chunk 显示编号和字符数
- 如果是 PDF 等分页文档，会显示页码

> **什么是 Chunk？**
> 长文档会被自动切分成多个小块（Chunk），用于向量化存储和检索。每个 Chunk 通常是几百字的一段文本。

### 5. 智能问答

**访问地址**: http://localhost:3000/chat

这是系统的核心功能，基于知识库内容进行 AI 问答。

#### 5.1 界面布局

```
┌─────────────────────────────────────────────────────────┐
│  知识库选择  │  对话列表        │   聊天区域              │
│  (左侧)      │  (中间左)       │   (右侧)                │
└─────────────────────────────────────────────────────────┘
```

#### 5.2 选择知识库

1. 在左上角的下拉框中选择一个知识库
2. 问答会基于该知识库的内容进行

> 💡 不同的知识库对应不同的知识领域，提问前请确保选择了正确的知识库。

#### 5.3 新建对话

- 点击「**+ 新对话**」按钮
- 会创建一个新的空白对话
- 也可以直接提问，系统会自动创建对话

#### 5.4 对话列表

- 显示所有历史对话
- 点击可以切换对话
- Hover 时右侧显示删除按钮
- 顶部显示对话总数

**删除对话**:
1. 将鼠标悬停在对话项上
2. 点击右侧的垃圾桶图标
3. 确认删除

#### 5.5 发送问题

1. 在底部输入框中输入问题
2. 按 `Enter` 发送，或点击「发送」按钮
3. 按 `Shift + Enter` 换行

**支持的操作**:
- ✅ 纯文本提问
- ✅ 多行输入（Shift+Enter）
- ✅ 流式输出（AI 回复逐字显示）
- ✅ 思考中动画（三色跳动点）

#### 5.6 消息展示

**用户消息（右侧）**
- 紫色渐变气泡
- 橙色头像
- 位于聊天区域右侧

**AI 消息（左侧）**
- 玻璃态卡片样式
- 蓝紫色头像
- 位于聊天区域左侧
- 可能包含引用来源

#### 5.7 引用来源

AI 回答下方可能会显示「引用来源」区域：

- 引用来源用数字标记（[1] [2] [3]）
- 每个引用显示文档名称
- 最多显示 3 个最相关的来源
- 点击数字可以快速定位来源文档

> 💡 引用来源是 RAG 技术的核心特性，让你知道 AI 的回答来自哪些文档，增加可信度。

### 6. 检索功能

**访问地址**: http://localhost:3000/search

不经过 AI 生成，直接在知识库中搜索相关内容片段。

#### 6.1 使用方法

1. 在搜索框中输入关键词
2. 点击「搜索」按钮或按 Enter
3. 查看搜索结果列表

#### 6.2 搜索结果

每条结果包含：

| 信息 | 说明 |
|------|------|
| **文档名称** | 内容来源的文档 |
| **相似度** | 与查询的匹配程度（百分比） |
| **内容片段** | 相关的文本内容 |
| **来源类型** | 文本/网页等 |
| **页码** | 如果是分页文档，显示页码 |

#### 6.3 检索 vs 问答的区别

| 特性 | 检索 | 问答 |
|------|------|------|
| **结果形式** | 原文片段 | AI 生成的回答 |
| **AI 参与** | 否 | 是 |
| **速度** | 快 | 稍慢 |
| **可信度** | 100% 原文 | 需要验证引用 |
| **适用场景** | 找原文、查资料 | 获得答案、解释说明 |

## 常见问题

### Q1: 上传文档后多久可以用于问答？

A: 通常几秒钟到几分钟不等，取决于文档大小。文档状态显示「已完成」（绿色）后就可以用于问答了。

### Q2: 支持 PDF、Word 文档吗？

A: 当前版本支持 .txt、.md、.html 格式。PDF 和 Word 支持将在后续版本中添加。

### Q3: 为什么 AI 回答说「未找到相关内容」？

A: 可能的原因：
1. 知识库中没有相关文档 → 请先上传文档
2. 问题的表述与文档内容差异较大 → 尝试换一种问法
3. 文档还在处理中 → 等待文档状态变为「已完成」

### Q4: 如何修改密码？

A: 当前版本暂不支持自助修改密码，请联系管理员。

### Q5: 对话会保存多久？

A: 对话会永久保存，除非你手动删除。你可以在对话列表中随时查看历史对话。

### Q6: 如何退出登录？

A: 点击侧边栏底部用户信息区域右侧的「登出」按钮（箭头图标）。

### Q7: 知识库的「公开」和「私有」有什么区别？

A: 
- **私有**：只有创建者本人可以看到和使用
- **公开**：所有登录用户都可以看到和使用，但只有创建者可以管理（上传/删除文档）
