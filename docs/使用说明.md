# RAG 知识库问答系统使用说明

## 1. 系统简介

本项目是一个基于 RAG（Retrieval-Augmented Generation，检索增强生成）的知识库问答系统，支持：

- 知识库创建与管理
- 文档上传、解析、预览与重新处理
- 基于知识库内容的智能问答
- 搜索结果与回答引用来源展示
- 多轮对话与对话历史保存

适合用于：

- 企业内部知识库问答
- 培训资料问答
- 项目文档检索与总结
- 教学课件内容检索

## 2. 访问地址

本地启动后可访问：

- 前端：http://127.0.0.1:3000
- 后端健康检查：http://127.0.0.1:8000/api/v1/health

## 3. 默认登录账号

默认管理员账号：

| 项目 | 值 |
| --- | --- |
| 邮箱 | `admin@example.com` |
| 密码 | `Admin123!` |

## 4. 首次使用流程

推荐按以下顺序体验：

1. 登录系统
2. 创建知识库
3. 上传文档
4. 等待文档处理完成
5. 进入聊天页开始提问
6. 查看回答下方的引用来源

## 5. 功能说明

### 5.1 登录页

地址：`/login`

功能：

- 输入邮箱和密码登录
- 登录成功后自动跳转到首页
- 登录状态保存在本地

### 5.2 首页

地址：`/`

功能：

- 展示欢迎信息
- 快速进入知识库、聊天、搜索等页面
- 查看系统主要功能入口

### 5.3 知识库列表页

地址：`/kb`

功能：

- 查看已有知识库
- 创建知识库
- 编辑知识库名称、描述、可见性
- 删除知识库

### 5.4 知识库详情页

地址：`/kb/[id]`

功能：

- 上传文档
- 查看文档列表
- 查看文档状态：排队中、处理中、已完成、无可用内容、处理失败
- 查看文档预览
- 查看 Chunk 列表
- 重新解析文档
- 删除文档
- 对文本类文档进行在线编辑

### 5.5 聊天页

地址：`/chat`

功能：

- 选择知识库发起问答
- 支持流式回答
- 显示回答引用来源
- 支持折叠 / 展开引用来源列表
- 支持复制回答内容
- 支持对话历史切换与删除

### 5.6 搜索页

地址：`/search`

功能：

- 输入关键词进行知识库检索
- 支持按知识库过滤
- 查看命中的文档片段、页码与来源

## 6. 文档上传说明

当前支持的常见格式：

- `.txt`
- `.md`
- `.markdown`
- `.html`
- `.htm`
- `.pdf`

说明：

- 小文件通常同步处理
- 大文件或 PDF 可能进入异步处理队列
- 异步处理中可稍后刷新查看状态

## 7. 回答引用来源说明

聊天回答中的引用来源区域现在支持折叠展示：

- 默认只显示摘要，减少消息卡片高度
- 点击后可展开查看来源列表
- 每条来源可展示文档名和页码范围

这样在长回答场景下，界面会更紧凑。

## 8. 常见问题

### 8.1 页面打不开

先检查：

- 后端是否已启动
- 前端是否已启动
- `3000` 和 `8000` 端口是否被占用

### 8.2 登录失败

先确认：

- 使用的是默认账号 `admin@example.com / Admin123!`
- 后端启动时已成功初始化数据库

### 8.3 上传后文档一直没有内容

可能原因：

- 文档仍在处理中
- 文档未提取出有效文本
- 文档处理失败，需要重新解析

### 8.4 前端构建失败并提示 `.next` 文件占用

如果你正在运行 `npm run dev`，同时再执行 `npm run build`，可能会因为 `.next` 被占用报错。先关闭开发服务再重新构建即可。

## 9. 建议体验路径

如果你是第一次看这个项目，推荐体验顺序：

1. 登录
2. 创建一个知识库
3. 上传一份文本或 PDF
4. 等待处理完成
5. 打开聊天页提问
6. 展开引用来源查看引用明细
